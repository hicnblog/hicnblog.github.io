Apa itu Markdown

Markdown ialah cara untuk memberikan styling pada teks di web. Styling yang dimaksud ialah merubah tampilan dokumen yaitu :

  • Memformat kata tebal atau miring.
  • Menambahkan gambar.
  • Membuat daftar list.
  • Membuat Link.
  • Dan lain-lain.

Markdown hanya menggunakan teks biasa, yang sebagian besar terdiri dari karakter non-abjad seperti * atau #.

Anda dapat menggunakan Markdown dengan mudah pada Github seperti pada Gist, Komentar Issue atau Pull Request dan file dengan ekstensi .md atau .markdown. Tetapi tidak hanya di Github saja yang bisa menggunakan Markdown bisa di linkungan-lingkungan pemrograman yang lain. Langsung cek di Google saja ya guys .

Panduan singkat dalam penggunaan Markdown :

Bold/italic

Lists

Images

Links

Blockquotes

Syntax highlighting

Tables

Emoji
